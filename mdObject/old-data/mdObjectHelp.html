GE Centricity $mdObject Help

The $mdObject is object oriented JavaScript GE Centricity HTML Form component.

Patient Object: $mdObject.patient

The $mdObject.patient.patientId returns the patient’s ID number.

The $mdObject.patient.pid returns the internal PID number for patient record.

The $mdObject.patient.medicalRecordId returns the patient’s medical record number.

The $mdObject.patient.externalId returns the patient’s ID from an external system, such as a billing or lab system.

The $mdObject.patient.printId returns the preferred patient ID number for printed materials.

The $mdObject.patient.ssn returns the patient’s Social Security number.

The $mdObject.patient.firstName the patient’s first name.

The $mdObject.patient.middleName the patient’s middle name.

The $mdObject.patient.lastName the patient’s last name.

The $mdObject.patient.labelName returns the patient’s full name formatted as follows: title first middle last, suffix.

The $mdObject.patient.namePrefix returns the patient’s preferred title. for example: Mr., Mrs., Ms., Miss, Dr.

The $mdObject.patient.nameSuffix returns the title used after the patient’s name. For example: MD, Ph.D..

The $mdObject.patient.sex returns the patient’s sex.

The $mdObject.patient.address returns patient address object.

The $mdObject.patient.address returns patient's address object.

The $mdObject.patient.race returns patient's race.

The $mdObject.patient.ethnicity returns patient's ethnicity.

The $mdObject.patient.dateOfBirth returns patient's date of birth. The value is in MM/DD/YYYY format.

The $mdObject.patient.dateOfDeath returns patient's date of death. The value is in MM/DD/YYYY format.

The $mdObject.patient.maritalStatus returns the patient’s marital status (Single, Married, Divorced, Widowed, Separated, Other or Undetermined).

The $mdObject.patient.language returns the patient’s preferred language.

The $mdObject.patient.email returns the patient’s e-mail address.

The $mdObject.patient.phone returns the patient’s phone object.

The $mdObject.patient.contactBy returns the patient’s contact by information.

The $mdObject.patient.location returns the array of patient’s Location objects.

The $mdObject.patient.employmentStatus returns the patient’s employment status.

The $mdObject.patient.clinicStatus returns the patient’s current status in the clinic.

The $mdObject.patient.primaryCarePhysicianName returns the name of the primary care physician.

The $mdObject.patient.referringProvider returns the patient’s referring provider object.

The $mdObject.patient.observations returns the array of patient’s observations. Usage: $mdObject.patient.observations('weight') return array of Observation object containing the patient's weight. The unsigned, current chart observations are first in the list. The observations are signed and unsigned in the current document.

The $mdObject.patient.insurances returns the patient’s insurances array. The array contains 3 Insurance objects: primary, secondary and third insurance.

The $mdObject.patient.immunizations returns the array of patient’s immunizations. The array contains Immunization objects.

The $mdObject.patient.carePlans returns the array of patient’s care plans. The array contains CarePlan objects.

The $mdObject.patient.allergies returns the array of patient’s allergies. Rework in next release.

The $mdObject.patient.medications returns the patient’s medications. Not implemented.

The $mdObject.patient.problems returns the patient’s problems. Not implemented.

The $mdObject.patient.orders returns the patient’s orders. Not implemented.

The $mdObject.patient.protocols returns the patient’s protocols. Not implemented.

The $mdObject.patient.directives returns the patient’s orders. Not implemented.
